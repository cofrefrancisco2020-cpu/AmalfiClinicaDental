# Amalfi Clínica Dental - versión 1

Página web estática creada para Amalfi Clínica Dental a partir del material visible en Instagram y capturas entregadas.

## Cómo abrir

Abrir `index.html` en el navegador o usar un servidor local desde esta carpeta.

## Archivos principales

- `index.html`: estructura de la página.
- `styles.css`: diseño responsive y animaciones.
- `script.js`: menú móvil, aparición de secciones y formulario hacia WhatsApp.
- `brief-marca.md`: investigación, decisiones y limitaciones.
- `assets/`: imágenes y logo finales usados por la página.

## Assets

Todos los recursos finales están en `assets`, sin subcarpetas.

- Logo recreado como SVG limpio a partir del logo visible en Instagram.
- Retratos del equipo creados con IA usando las capturas como referencia, sin interfaz ni textos incrustados.
- Imagen hero de clínica dental generada con IA siguiendo la paleta turquesa/blanca de Amalfi.
- Imagen educativa de cepillado generada con IA sin letras.

## Limitaciones de investigación

Instagram bloqueó la revisión directa desde el entorno de trabajo. La versión se basó en las capturas entregadas por el usuario y los datos visibles en ellas. La captura de fachada se usó solo para confirmar ubicación, no como imagen final.
